## For AI Agents

### Conventions

- Bash scripts: ShellCheck-clean. Run `./scripts/lint-shell.sh` before committing.
- Python scripts: PEP 8. Use `os.getenv()` for all credentials — no hardcoded values.
- Dashboard scripts read from `~/Desktop/.env` for API keys (not committed).
- `claudesec-dashboard.html` (the security-scan dashboard) plus `scan.html` /
  `scan-report.json` are generated artifacts in `.gitignore` — regenerated by
  `dashboard-refresh.yml` (a repo self-scan; no external API keys needed).
- `claudesec-asset-dashboard.html` is **committed on purpose** (it is NOT
  gitignored): `build-dashboard.py` needs operator secrets (Datadog / Notion /
  Jamf / Google Sheets) and produces org-specific asset data, so it is not
  auto-regenerated in public CI. An operator rebuilds it locally
  (`python3 scripts/build-dashboard.py`) and commits the refresh.

### Running the Dashboard

```bash
# Safe mode (auto port fallback, no Docker required)
./scripts/run-dashboard-safe.sh

# Kill stale process and serve
./scripts/run-dashboard-safe.sh --kill-port

# Docker mode
./scripts/run-dashboard-docker.sh

# Build only, no serve
./scripts/run-full-dashboard.sh --no-serve
```

Dashboard served at `http://localhost:11777/`.
