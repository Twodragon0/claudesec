# ISO 27001:2022

ISO 27001:2022 is the latest revision of the international standard for Information Security Management Systems (ISMS). Major restructuring: 114 controls in 14 domains (2013) → **93 controls in 4 themes** (2022).

## Key Changes from 2013

| Aspect | ISO 27001:2013 | ISO 27001:2022 |
|--------|----------------|----------------|
| Annex A controls | 114 | 93 |
| Control domains | 14 | 4 themes |
| New controls | — | 11 new controls |
| Merged controls | — | 24 merged (from 57) |
| Attributes | Not used | 5 attributes per control |
| Transition deadline | — | October 31, 2025 |

## Control Attributes (New in 2022)

Each control now has 5 attributes for filtering and prioritization:

| Attribute | Values |
|-----------|--------|
| **Control type** | Preventive, Detective, Corrective |
| **Information security properties** | Confidentiality, Integrity, Availability |
| **Cybersecurity concepts** | Identify, Protect, Detect, Respond, Recover |
| **Operational capabilities** | Governance, Asset mgmt, Protection, etc. |
| **Security domains** | Governance & ecosystem, Protection, Defence, Resilience |
